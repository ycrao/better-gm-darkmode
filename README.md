better-gm-darkmode
------------------

>  更好的夜间模式助手，适配不支持暗黑模式的网站。

### 修改说明

原始油猴脚本文件位于 [油小猴](https://www.youxiaohou.com/tool/install-darkmode.html)，本油猴脚本主要优化改动点：

改黑名单为白名单（即 “排除以下网站” 修改为 “仅在下列网站启用夜间模式”）。这样子可以避免掉一些已经支持暗黑模式的网站不会与本油猴脚本的暗黑模式产生冲突。

优化之后的行为如下：

- 新开的网站默认不启用暗黑模式
- 不支持暗黑的网站需要手动添加到白名单
- 支持原生暗黑的网站不会被覆盖

[安装 better-gm-darkmode 油猴脚本](https://raw.githubusercontent.com/ycrao/better-gm-darkmode/refs/heads/main/darkmode.user.js)
